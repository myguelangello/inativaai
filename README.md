# Inativa Aí

### API para inativação segura de colaboradores desligados no período definido

### Tecnologias utilizadas:
- Flask: 3.1.2
- oracledb: 3.4.0
- psycopg2: 2.9.11
- SQLAlchemy: 2.0.44

### Configuração do ambiente virtual:
Para criar um ambiente virtual, execute os seguintes comandos:
```bash
  python -m venv .venv
```


Ativar ambiente virtual no Windows (sempre que for executar o projeto):
```bash
  .venv\Scripts\activate
```

Instalar dependências:
```bash
  pip install -r requirements.txt
```

### Executar a aplicação:
```bash
  python app.py
```
